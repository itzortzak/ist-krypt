# Istorika Kryptografimata v18

Εκπαιδευτικό web app για GitHub Pages με κρυπτογραφήματα Ιστορίας Προσανατολισμού.

## Περιεχόμενο
- 5 βασικά κεφάλαια
- 88 κρυπτογραφήματα
- φίλτρα κεφαλαίου, υποενότητας, τύπου και δυσκολίας
- υποδείξεις, αποκάλυψη γράμματος, στατιστικά προόδου

## Αρχεία
- index.html
- instructions.html
- info.html
- style.css
- app.js
- data/puzzles.json


## Κατάσταση αξιοπιστίας
Η v19 είναι ώριμη και λειτουργική για δοκιμή και πιλοτική χρήση. Για πλήρη εκπαιδευτική αξιοπιστία συνιστάται τελικός χειροκίνητος έλεγχος όλων των λημμάτων, υποδείξεων και επεξηγήσεων σε αντιπαραβολή με το σχολικό βιβλίο.
